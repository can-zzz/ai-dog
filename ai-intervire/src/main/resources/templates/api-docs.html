AI智能助手 RESTful API 接口文档

概览 聊天接口 历史记录 统计信息 WebSocket 健康检查

概览

AI智能助手提供了完整的RESTful API和WebSocket接口，支持：

  • 智能对话聊天
  • 对话历史管理
  • 实时WebSocket通信
  • 会话统计信息

基础URL

http://localhost:8081/api/ai

请求格式

所有POST请求都使用JSON格式，请设置Content-Type为application/json

响应格式

所有响应都是JSON格式，包含success字段表示请求是否成功

聊天接口

POST /api/ai/chat

发送消息给AI助手并获取回复。

请求参数

{ "message": "你好，请介绍一下自己", "sessionId": "session-123", "modelType": "qwen", "saveHistory": true }

参数说明

参数 类型 必填 说明
message string 是 用户消息内容，最大2000字符
sessionId string 否 会话ID，不提供会自动生成
modelType string 否 模型类型，默认为qwen
saveHistory boolean 否 是否保存对话历史，默认true

响应示例

{ "message": "你好！我是AI智能助手，很高兴为您服务...", "sessionId": "session-123", "timestamp": "2024-01-01T12:00:00", "model": "qwen", "processingTime": 1500, "success": true, "error": null }

历史记录

GET /api/ai/history/{sessionId}

获取指定会话的历史记录。

响应示例

[ { "content": "你好", "type": "USER", "sender": "用户", "sessionId": "session-123", "timestamp": "2024-01-01T12:00:00", "messageId": "msg-001" }, { "content": "你好！我是AI智能助手", "type": "ASSISTANT", "sender": "AI助手", "sessionId": "session-123", "timestamp": "2024-01-01T12:00:01", "messageId": "msg-002" } ]
DELETE /api/ai/history/{sessionId}

清除指定会话的历史记录。

响应示例

{ "success": true, "message": "会话历史已清除", "sessionId": "session-123" }

统计信息

GET /api/ai/stats/{sessionId}

获取指定会话的统计信息。

响应示例

{ "sessionId": "session-123", "messageCount": 10, "model": "qwen" }

WebSocket接口

连接地址

ws://localhost:8081/ws

支持的消息类型

  • /app/chat - 发送聊天消息
  • /app/connect - 用户连接通知
  • /app/disconnect - 用户断开连接通知

订阅频道

  • /topic/messages/{sessionId} - 接收特定会话的消息
  • /topic/users - 接收用户连接状态变化

消息格式

{ "content": "消息内容", "type": "USER|ASSISTANT|SYSTEM", "sender": "发送者", "sessionId": "会话ID", "timestamp": "时间戳", "messageId": "消息ID" }

JavaScript示例

const socket = new SockJS('/ws'); const stompClient = new StompJs.Client({ webSocketFactory: () => socket, onConnect: () => { stompClient.subscribe('/topic/messages/session-123', (message) => { const data = JSON.parse(message.body); console.log('收到消息:', data); }); } }); stompClient.activate(); // 发送消息 stompClient.publish({ destination: '/app/chat', body: JSON.stringify({ content: '你好', type: 'USER', sessionId: 'session-123' }) });

健康检查

GET /api/ai/health

检查服务健康状态。

响应示例

{ "status": "UP", "service": "AI Assistant", "timestamp": 1704096000000 }

错误码说明

HTTP状态码 说明 示例
200 请求成功 正常响应
400 请求参数错误 消息内容为空或过长
500 服务器内部错误 AI模型调用失败
